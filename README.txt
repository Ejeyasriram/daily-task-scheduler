Daily Task Scheduler Project

Files Included:
1. index.html - Main webpage
2. style.css - Styling for the application
3. script.js - JavaScript functionality

How to Run:
1. Extract the ZIP file.
2. Open index.html in any web browser.
3. Add, complete, and delete tasks.

Features:
- Add tasks with time
- Mark tasks as completed
- Delete tasks
- Simple and responsive UI
